Добрый день!

Для запуска приложения следуйте дальнейшей инструкции:
1. Склонируйте проект из репозитория
2. Подтяните все необходимые зависимости и соберите проект
3. Запустите локальной контейнер файл docker-compose.yml
4. Запустите приложение (запускается на порту 8080)
5. По-умолчанию создается 2 тестовых пользователя admin с паролем admin и user с паролем user (роли соответствуют их именам)
6. Для подключения к базе данных используйте креды (POSTGRES_USER: postgres POSTGRES_PASSWORD: postgres POSTGRES_DB: bank_rest)
7. Можете использовать как Postman, так и Swagger UI http://localhost:8080/swagger-ui/index.html
8. swagger интегрирована возможность авторизации, для этого:
8.1. Воспользуйтесь EP авторизации /auth/signin и получите токен
8.2. В правом верхнем углу нажмите "Authorize" и введите токен
8.3. Теперь можно использовать все возможности приложения (согласно роли)
9. Все контроллеры и EP прописаны в документации
10. В случае большинства ошибок приложение в теле ответа будет присылать причину ошибки

Спасибо за внимание!
